## 💡 KEY FEATURES DEEP DIVE

### Progress Bar Algorithm
```javascript
progressPercentage = ((currentRound - 1) / totalRounds) * 100
// Example: Round 2 of 4 = (1/4) * 100 = 25%
// Example: Round 4 of 4 = (3/4) * 100 = 75%
```

### Toast Auto-Dismiss
```javascript
setShowCelebration(true);           // Show
setTimeout(() => 
  setShowCelebration(false), 2000   // Hide after 2s
);
```

### Confetti Conditions
```javascript
if (playerSelectedVote === revealedLie) {
  showConfetti(); // Only for CORRECT answers
}
```

### Staggered Animation
```javascript
transition={{ delay: i * 0.1 }}
// Item 0: 0ms delay
// Item 1: 100ms delay
// Item 2: 200ms delay
```
